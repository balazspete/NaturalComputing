Thanks for helping out! Well, what is this website about?

The aim is to generate different colour-sets, which are used in websites.

Because computers cannot tell what colour sets are nice and what aren't, we need to teach them, but for that a large amount of data needs to be collected.

This is where you come in. If you have a spare minute two, I'd like you to randomly generate some colour sets by using the random button. You can ajust and modify the colours by clicking on the colours themselves in the sidebar.

Once you found a colour-set that you like, click the like button. If there's a set that you do not fancy, click dislike. That's all :)
