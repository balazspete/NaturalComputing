I have a project in which I need to be able to tell which sets of colours are nice, and which aren't. But I'd need to do it thousands of times per experiment, for hundreds of experiments. So i kinda have to use a computer...

Computers cannot tell which colour sets are nice and which aren't, so we need to teach them. To make sure they learn the right thing, a large amount of data needs to be collected.

This is where you come in. The aim of this website is to collect data. If you have a spare minute or two, I'd like you to randomly generate some sets of colours, you can do this by using the random button. You can ajust and modify the colours by clicking on the coloured boxes in the sidebar.

Once you found a colour set that you like, click the like button. If there's a set that you do not fancy, click dislike. That's all :)

As more data is collected from different souces, the resulting system will become more reliable.
